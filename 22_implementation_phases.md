# 22. Implementation Phases

## Phase 1: Foundation (MVP) - 4-6 Weeks
- **Core:** Supabase Setup + Auth.
- **AI:** Basic Gemini Page Parser (Vision to JSON).
- **UI:** Teacher Dashboard (Upload) and Classroom Board (Simple Slides).
- **Gamification:** Basic Wheel and Star Counter.

## Phase 2: Interactivity & Growth - 6-8 Weeks
- **Classroom:** Hotspot Interactive Scenes + Team Battle Games.
- **Mobile:** Student PWA Home App launch (Map, Scrambles).
- **AI:** Real-time pronunciation grading integration.
- **Social:** Story Dubbing feature (MVP).

## Phase 3: The "Magic" Upgrade - 4-6 Weeks
- **Scanner:** Context-aware recurring character detection.
- **Assets:** Nano Banana (AI Generation / Search) integration.
- **Avatar:** Hybrid SVG System with AI Accessories.
- **Parent:** Reporting dashboard and sharing features.

## Phase 4: Polish & Scale
- **Tech:** WebSockets optimization for <50ms latency.
- **Content:** Expanding Game Template library.
- **Design:** Full micro-animation and sound effect pass.
